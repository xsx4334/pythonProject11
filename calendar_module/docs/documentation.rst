Documentation
=============

.. currentmodule:: tkcalendar

.. note:: 

    This documentation only addresses the methods specific to the :class:`Calendar`
    and :class:`DateEntry` widgets, but these widgets also possess the `methods common to
    all tkinter widgets <http://infohost.nmt.edu/tcc/help/pubs/tkinter/web/universal.html>`_.
    and the `one common to all ttk widgets <http://infohost.nmt.edu/tcc/help/pubs/tkinter/web/ttk-Widget.html>`_.
    In addition, the :class:`DateEntry` also have all the `methods of the Entry widget  <http://infohost.nmt.edu/tcc/help/pubs/tkinter/web/entry.html>`_.
